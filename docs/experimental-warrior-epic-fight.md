# Warrior Epic Fight Experiment

## Objetivo

Probar si Nexus Realms puede tener un rol Guerrero estilo Elden Ring/NightfallCraft con habilidades epicas, combos y skill tree.

## Mods de prueba

- Epic Fight
- EpicFight-Nightfall
- Epic Fight: Skill Tree

## Riesgos

- Puede chocar con Better Combat.
- Puede chocar con Combat Roll.
- Puede chocar con PlayerAnimator/Fresh Animations.
- Puede afectar TaCZ/Punchy/Shoulder Surfing.
- Puede cambiar demasiado el combate base del pack.
- No se debe mergear a dev hasta probarlo muy bien.

## Pruebas en Prism

1. Arrancar cliente.
2. Crear mundo creativo nuevo.
3. Probar espada vanilla.
4. Probar hacha vanilla.
5. Probar arma de Simply Swords.
6. Probar cambios de postura/modo de Epic Fight.
7. Abrir Skill Tree.
8. Probar habilidades de Nightfall.
9. Probar Combat Roll.
10. Probar Better Combat.
11. Probar TaCZ en primera persona.
12. Probar TaCZ con Shoulder Surfing.
13. Probar Punchy con herramientas normales.
14. Probar Iron's Spells.
15. Probar un boss pequeno.
16. Revisar crasheos, animaciones rotas, keybinds duplicados y FPS.

## Criterio de exito

- Arranca sin crash.
- El combate melee se ve claramente mejor para Guerrero.
- TaCZ sigue funcionando.
- Punchy sigue funcionando.
- Shoulder Surfing sigue funcionando.
- Iron's Spells sigue funcionando.
- No hay doble render raro.
- No hay conflictos graves de keybinds.
- No baja demasiado el rendimiento.

## Criterio de rechazo

- Crashea al arrancar.
- Rompe TaCZ/Punchy.
- Rompe Shoulder Surfing.
- Rompe Iron's Spells.
- Hace incompatible Better Combat de forma imposible de arreglar.
- Genera animaciones del jugador demasiado rotas.
