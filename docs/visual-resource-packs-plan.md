# Pack 18.0 - Visual Resource Packs and Fresh Animations

## Scope

Pack 18.0 adds the visual foundation for Nexus Realms:

- Entity Model Features
- Entity Texture Features
- Entity Culling
- Fresh Animations
- Fresh Animations addons
- LowOnFire
- 3D item/block resource packs

## Not in scope

The following are not part of Pack 18.0:

- Particle Rain
- Cool Rain
- Serene Seasons
- One-Sleep Timelapse
- Loot Journal
- Legendary Tooltips
- Sophisticated Backpacks
- Factions
- Bosses
- Dungeons
- Worldgen
- Tough As Nails

Particle Rain and weather-related mods belong to Pack 24.0.

Loot UI and inventory mods belong to Pack 19.0.

## Compatibility rules

- Do not install OptiFine.
- Do not install Sinytra Connector.
- Do not install Fabric-only packs or mods.
- Do not install NeoForge-only files.
- Do not commit root `options.txt`.
- Do not commit manually downloaded `.zip` files.
- Use packwiz-managed metadata whenever possible.

## Expected visual result

Nexus Realms should feel more alive and polished:

- Better animated mobs.
- Better third-person player presentation.
- Better visual items and blocks.
- Lower first-person fire.
- More premium RPG visual identity.
