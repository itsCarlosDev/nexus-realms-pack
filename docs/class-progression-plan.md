# Nexus Realms Class Progression Plan

## Objetivo

Definir progresion inicial por clase usando FTB Quests como frontend de misiones/recompensas y KubeJS como fuente de verdad de clase.

## Principio

- KubeJS decide la clase.
- FTB Quests no debe cambiar la clase directamente.
- FTB Quests puede leer o depender de progreso, tags o comandos.
- Los kits iniciales siguen en KubeJS.
- Las recompensas avanzadas podran venir de FTB Quests o KubeJS.

## Guerrero

Identidad:

- combate cuerpo a cuerpo;
- armas pesadas;
- Epic Fight;
- Simply Swords.

Progresion inicial:

1. Elegir Guerrero.
2. Recibir arma inicial.
3. Derrotar mobs basicos.
4. Usar arma Simply Swords.
5. Desbloquear o explicar Skill Tree.
6. Primer boss menor.

Recompensas futuras:

- comida;
- hierro/acero;
- armas mejores;
- puntos/progresion de skill si el mod lo permite;
- acceso a arenas/bosses.

## Mago

Identidad:

- hechizos;
- libro magico;
- recursos arcanos.

Progresion inicial:

1. Elegir Mago.
2. Recibir Copper Spell Book.
3. Lanzar primer hechizo.
4. Conseguir amethyst shards.
5. Mejorar libro/hechizos.
6. Derrotar enemigo usando magia.

Recompensas futuras:

- tinta/pergaminos si aplica;
- materiales magicos;
- scrolls;
- upgrade de spellbook;
- acceso a mobs magicos.

## Pistolero

Identidad:

- armas de fuego;
- municion;
- supervivencia tactica.

Progresion inicial:

1. Elegir Pistolero.
2. Recibir Glock 17.
3. Recibir municion 9mm.
4. Aprender a apuntar/recargar.
5. Craftear o conseguir mas municion.
6. Derrotar enemigo a distancia.

Recompensas futuras:

- municion;
- attachments;
- polvora;
- recursos de mantenimiento;
- acceso a armas mejores.
